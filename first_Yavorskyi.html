Гіпертекст

Класичне поняття «гіпертекст» – (від англ. hypertext) – це документ, що містить гіперпосилання на інші документи, які можуть бути відображені безпосередньо з вихідного (первинного) документа через активацію гіперпосилання.

Відповідно гіперпосилання – це адреса іншого мережевого інформаційного ресурсу, що вміщує документ, на який бажає перейти користувач.

Для нормальної роботи – навігації у віртуальному середовищі мережі Internet допомагає спеціально розроблене програмне забезпечення – веб- оглядачі / веб-навігатори / веб-браузери (Netscape Navigator, Internet Explorer, Mozila FireFox, Opera, Google Chrome тощо).

1 html відсутні Є контейнером для решти тегів та всього коду сторінки
2 head відсутні Тег який зберігає метаданні документа(іконка, мова, заголовок, назва)
3 title Основні: title, translate, accesskey, class, dropzone, hidden, id, lang, style Вказує заголовок сторінки
4 body Основні: title, translate, class, color, style, id Містить у собі основну структуру сторінки, "тіло"
5 p Основні: class, font, style Створення абзацу в тексті
6 hr Основні: align, color, class, noshadow, size, width Створення горизонтальної лінії "роздільника"
7 h1, h2, h3... Основні: align, class, style Визначення заголовків тексту від найбільш важливого до найменш важливого
8 br відсутні Символ абзацу, перенесення тексту на нову стрічку
9 div, span Усі загальні параметри Теги "контейнери". Зберігають у собі об'єкти/текст для наступних маніпуляцій. "div" - контейнер для об'єктів, "span" - контейнер для тексту
10 big, small відсутні Модифікатори розміру тексту
11 style media, scroped, type Містить у собі CSS код, модифікатор для HTML-коду
12 table, tr, td Модифікується через CSS(style, class) Елементи таблиці.
table - тіло таблиці
tr - рядок таблиці
td - комірка в рядку
